# app_llm_prompts

このディレクトリは、アプリケーションが実行時に使用するプロンプトのためのものです。

これらのプロンプトは、アプリケーションが使用する Gemini または他の文書解析 LLM 向けです。

Claude Code の指示ではありません。

Claude Code の指示は以下に記載します：

- `CLAUDE.md`
- cc-sdd が生成した成果物
- 将来的に明示的に導入する場合は `.claude/` 設定

## 初期ファイル

```text
naming_policy.draft.md
extraction_policy.draft.md
```

これらはドラフトです。

最終的なプロンプト構造は実装とテストの中で決定します。

## 設計意図

人間がメンテナンスできるポリシーと例に Markdown を使用します。

命名の挙動に複雑な YAML ルールシステムを避けます。

以下のような機械的な安全ルールはコードで実施します：

- ファイル名のサニタイズ
- 重複時のサフィックス付与
- 長さの制限
- 成功/要確認/エラーのプレフィックス処理
- JSON スキーマのバリデーション
- 信頼度しきい値の処理
